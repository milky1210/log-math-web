# 📊 Logarithmetic (Log-Zan)

**対数で世界を見よう！ログ算で直感的な数値感覚を養う教育Webアプリケーション**

## 🎯 プロジェクト概要

Logarithmetic (Log-Zan) は、対数（Logarithm）を使った計算手法「Log算」を学習・実践できる教育プラットフォームです。フェルミ推定や物理計算を直感的に行うスキルをゲーム感覚で身につけることができます。

### 主な特徴

- 📚 **学習モード**: スライド形式でLog算の基礎を学べる
- 🎮 **実戦モード**: レベル別のクイズで実践的なスキルを習得
- 📖 **定数表**: よく使う定数をカテゴリ別に管理・検索
- 🏆 **進捗管理**: 解いた問題数に応じてレベルアップ
- 💾 **ローカル保存**: 進捗状況はLocalStorageに自動保存

## 🚀 技術スタック

- **フレームワーク**: React 18 (Vite)
- **スタイリング**: Tailwind CSS
- **アイコン**: Lucide React
- **状態管理**: React Context API
- **データ永続化**: localStorage

## 🎓 Log算とは？

対数（Logarithm）を使って、巨大な数や複雑な計算を簡単に扱う手法です。

### 基本原理

- **掛け算 → 足し算**: \`log(A × B) = log(A) + log(B)\`
- **割り算 → 引き算**: \`log(A ÷ B) = log(A) - log(B)\`
- **累乗 → 掛け算**: \`log(A^n) = n × log(A)\`

### 覚えるべき基本定数

| 数値 | Log値 | 説明 |
|------|-------|------|
| 2 | 0.3 | 最重要！ |
| 3 | 0.48 | 約0.5 |
| 5 | 0.7 | = 10/2 |
| 10 | 1.0 | 基準 |
